Cloud / DevOps / DevSecOps Project

Multi-Container Docker Application

This project demonstrates a real-world containerized application architecture using Docker Compose. It separates the frontend, backend API, and database into independent services that run together as one system.

Frontend

Served with NGINX and exposed to the browser as the user-facing layer.

Backend API

Built with Node.js and Express to handle application logic and API requests.

Database

MongoDB runs as a separate container to represent the application data layer.

Orchestration

Docker Compose manages all services, networks, builds, and container startup.

Browser → Frontend Container → Backend API Container → MongoDB Container
GitHub → Docker Compose → Local / Cloud Deployment
